Web Platform discrepancies

A number of implementations support the following non-standard methods for accessors:

  • __lookupGetter__

    see also https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Object/LookupGetter

  • __lookupSetter__

    see also https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Object/LookupSetter

  • __defineGetter__

    see also https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Object/defineGetter

  • __defineSetter__

    see also https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Object/defineSetter
